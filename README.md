# 🚀 Database Mastery Serisi

Bu repository, farklı veritabanı teknolojileri kullanılarak geliştirilmiş projeleri içeren **Database Mastery** eğitim serisinin bir parçasıdır.
Amaç; farklı veritabanı yaklaşımlarını (SQL & NoSQL) gerçek senaryolar üzerinden deneyimlemek ve karşılaştırmalı olarak öğrenmektir.

---

## 📦 Mevcut Projeler

### 🟢 1. Transport MongoDb

📁 `DatabaseMastery.TransportMongoDb`

MongoDB kullanılarak geliştirilmiş bir **lojistik ve kargo takip sistemi**.
NoSQL veri modelleme yaklaşımı ile dinamik ve esnek veri yönetimi hedeflenmiştir.

---

### 🟢 2. HotCoffee PostgreSQL

📁 `DatabaseMastery.HotCoffeePostgreSql`

PostgreSQL kullanılarak geliştirilmiş bir **kafe/restaurant yönetim sistemi**.
İlişkisel veritabanı yapısı ile güçlü veri bütünlüğü ve sorgulama kabiliyeti sunar.

---

## 🧩 Yakında Eklenecek Projeler

🔜 Bu seri toplamda **3 farklı veritabanı teknolojisini** kapsayacak şekilde genişletilecektir.

Yakında eklenecek projeler ile birlikte:

* Farklı database türleri karşılaştırılacak
* Performans ve kullanım senaryoları analiz edilecek
* Gerçek dünya projeleri üzerinden deneyim kazanılacaktır

---

## 🛠️ Kullanılan Teknolojiler

* ASP.NET Core MVC
* C#
* MongoDB (NoSQL)
* PostgreSQL (Relational Database)
* Entity Framework Core
